# Koa2 Project

```bash
git clone https://github.com/LolipopJ/startup.git -b koa2 --depth 1
```

## Development

```bash
# Watch change of src/**/*.ts and config/**/*.ts
npm run dev
```

Open a new terminal, and run command:

```bash
npm run nodemon
```

## Deployment

```bash
# Run koa app as a pm2 instance
npm run deploy
```
